# app_gestion_salles

Nom : azzouz
Prénom : souhila

## Description

Ce projet est une application Python de gestion des salles.

Elle permet :

* d’ajouter une salle
* de modifier une salle
* de supprimer une salle
* de rechercher une salle
* d’afficher la liste des salles

## Technologies utilisées

* Python
* MySQL
* customTkinter
* Git / GitHub

## Objectif

Mettre en place une application avec une architecture en couches :

* Data (accès aux données)
* Models (modèles)
* Services (logique métier)
* Views (interface graphique)
